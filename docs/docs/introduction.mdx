# 介绍

**LangChain** 是一个用于开发大语言模型(LLMs)驱动的应用程序的框架。

LangChain 简化了 LLM 应用程序生命周期的每个阶段：
- **开发**：使用 LangChain 的开源[组件](/docs/concepts)和[第三方集成](/docs/integrations/providers/)构建应用程序。
使用 [LangGraph](/docs/concepts/architecture/#langgraph) 构建具有一流流式处理和人机协作支持的有状态代理。
- **生产化**：使用 [LangSmith](https://docs.smith.langchain.com/) 检查、监控和评估您的应用程序，以便持续优化并自信地部署。
- **部署**：使用 [LangGraph Platform](https://langchain-ai.github.io/langgraph/cloud/) 将您的 LangGraph 应用程序转换为生产就绪的 API 和助手。

## 架构

LangChain 框架由多个开源库组成。详细信息请参阅[架构](/docs/concepts/architecture/)页面。

- **`langchain-core`**：聊天模型和其他组件的基础抽象。
- **集成包**（如 `langchain-openai`、`langchain-anthropic` 等）：重要的集成已被拆分成轻量级包，由 LangChain 团队和集成开发者共同维护。
- **`langchain`**：构成应用程序认知架构的链式、代理和检索策略。
- **`langchain-community`**：由社区维护的第三方集成。
- **`langgraph`**：用于将 LangChain 组件组合成具有持久性、流式处理和其他关键特性的生产就绪应用程序的编排框架。参见 [LangGraph 文档](https://langchain-ai.github.io/langgraph/)。
