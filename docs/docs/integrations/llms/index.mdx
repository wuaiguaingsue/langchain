# LLMs（大语言模型）

:::caution
您当前正在查看记录[文本补全模型](/docs/concepts/text_llms)使用的页面。许多最新和最流行的模型是[聊天补全模型](/docs/concepts/chat_models)。

除非您特别使用更高级的提示技术，否则您可能要查找的是[这个页面](/docs/integrations/chat/)。
:::

[LLMs](/docs/concepts/text_llms)是接收字符串作为输入并返回字符串作为输出的语言模型。

:::info

如果您想编写自己的LLM，请参阅[此操作指南](/docs/how_to/custom_llm/)。
如果您想贡献一个集成，请参阅[贡献集成](/docs/contributing/how_to/integrations/)。

:::
