# 无效提示输入

[提示模板](/docs/concepts/prompt_templates)收到了缺失或无效的输入变量。

## 故障排除

以下方法可能有助于解决此错误：

- 仔细检查您的提示模板，确保其正确无误。
  - 如果您使用默认的f-string格式，并且在模板中的任何位置使用了花括号`{`，应该这样双重转义：`{{`（如果您想要渲染一个双花括号，则应使用四个花括号：`{{{{`）。
- 如果您使用的是[`MessagesPlaceholder`](/docs/concepts/prompt_templates/#messagesplaceholder)，确保您传入的是消息数组或消息类对象。
  - 如果您使用简写元组来声明提示模板，确保变量名用花括号包裹（`["placeholder", "{messages}"]`）。
- 尝试使用[LangSmith](https://docs.smith.langchain.com/)或日志语句查看提示模板的输入，确认它们符合预期。
- 如果您从[LangChain Prompt Hub](https://smith.langchain.com/prompts)获取提示，尝试提取并记录它，或者使用样本输入单独运行它，以确认它符合您的预期。
